    Topics in this project

    • Sync vs async — logs run immediately; timers & fetch run later
    • Event loop — call stack empties, then microtasks, then macrotasks
    • Callbacks — validateDuration uses a callback pattern
    • Promises — delay(), pending → fulfilled / rejected
    • Chaining — prepareCountdown() chains .then()
    • async / await — main countdown loop waits with await
    • try / catch — errors from validation & fetch are caught
    • fetch — loads a quote when the timer finishes
